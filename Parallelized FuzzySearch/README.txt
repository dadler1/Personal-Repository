README:

This document will specify instructions for running and compiling relevant scripts. NOTE: The presence of the input file is required.


To Compile and Run FuzzySearch:

>>python3 ./FuzzySearch.py input.txt <search keyword> <max levenshtein distance> <bash script boolean>

for more information, please review comments on the top of FuzzySearch.py



To Compile and Run FuzzySearchParallel:

>>python3 ./FuzzySearchParallel.py input.txt <search keyword> <max levenshtein distance> <bash script boolean> <number of threads>

for more information, please review comments on the top of FuzzySearchParallel.py
